# Jagarana with Shiva

Static one-page web app for Maha Shivaratri Jagarana, designed for simple hosting on Hostinger `public_html` (no Node.js runtime required).

## Stack
- HTML (`index.html`)
- CSS (`styles.css`)
- JavaScript (`app.js`, `hero.js`)
- Optional CDN scripts for Three.js hero rendering and jsPDF certificate generation

## Run Locally
```bash
cd /Users/tejash/Desktop/shivaratri-site
python3 -m http.server 8000
```
Open: `http://127.0.0.1:8000`
